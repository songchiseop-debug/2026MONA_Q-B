# 🦉 2026 모나용평 발왕산 여름 축제 · 퀴즈퍼즐 게임

발왕산 왕수리부엉이 FAMILY가 카테고리별로 문제를 내고, 정답을 맞혀 빙고판을 채우는 모바일 웹 게임입니다.
빙고판을 채울수록 뒤에 숨겨진 축제 이미지가 하나씩 드러나고, 16칸을 모두 완성하면 경품 응모 대상이 됩니다.

## 📁 파일 구성

```
index.html   ← 게임 본체 (단일 파일, 이미지·캐릭터 모두 내장)
README.md    ← 이 문서
```

- 부엉이 3D 캐릭터, 로고, 배경, 축제 이미지는 모두 `index.html` 안에 포함되어 있어 **별도 이미지 파일이 필요 없습니다.**
- 글꼴(Gothic A1)만 Google Fonts CDN에서 불러오므로 **실행 시 인터넷 연결이 필요합니다.**

## 🚀 GitHub Pages 배포 방법

1. GitHub에서 새 저장소(Repository)를 만듭니다. (예: `balwangsan-bingo`)
2. `index.html`과 `README.md`를 저장소에 업로드합니다. (드래그 앤 드롭 → Commit)
3. 저장소 상단 메뉴 **Settings → Pages** 로 이동합니다.
4. **Build and deployment → Source** 를 `Deploy from a branch` 로 두고,
   Branch를 **`main` / `(root)`** 으로 선택한 뒤 **Save**.
5. 잠시(1~2분) 기다리면 상단에 게임 주소가 생성됩니다.
   → `https://<사용자명>.github.io/<저장소명>/`

배포 후 스마트폰 브라우저에서 위 주소로 접속하면 바로 플레이할 수 있습니다.

## 💻 로컬에서 확인하기

`index.html` 파일을 더블클릭하면 브라우저에서 바로 실행됩니다. (인터넷 연결 필요)

## ✏️ 문제 · 카테고리 수정 방법

`index.html` 안 `<script>` 영역에서 수정합니다.

- **카테고리 이름 / 담당 캐릭터**: `const CATS = [...]` 배열의 `name` 값 (캐릭터 매핑은 `owl` 값).
- **문제**: `const QUESTIONS = [...]` 배열.
  - `c` : 카테고리 번호(0~3) — 어느 부엉이가 낼지 결정
  - `q` : 문제, `o` : 보기 4개, `a` : 정답 번호(0~3), `e` : 해설

## 🎮 게임 규칙

- 카테고리별 부엉이가 번갈아 출제 (30초 타이머, 뒤로 갈수록 짧아짐)
- 정답 → 원하는 빈 칸에 도장 → 그 칸의 축제 이미지 조각이 열림
- 오답 · 시간초과 → 목숨 1개 차감 (총 3개)
- 목숨 0 → 미션(구독·팔로우·공유) 인증 시 1회 재도전
- 16칸 완성 → 경품(숙박권 · 워터파크 이용권 등) 응모

## 🔗 카카오톡 공유 미리보기 (중요)

`index.html`과 같은 위치에 `og.jpg`(부엉이+퍼즐 미리보기 이미지)가 있어야 합니다.
카카오톡·문자로 링크를 보낼 때 미리보기 이미지가 안정적으로 뜨게 하려면,
`index.html` 상단의 `og:image`를 **전체 주소**로 바꿔주세요.

예) 배포 주소가 `https://myid.github.io/balwangsan/` 이면
```html
<meta property="og:image" content="https://myid.github.io/balwangsan/og.jpg"/>
```
- 카카오톡은 미리보기를 캐시합니다. 안 바뀌면 카카오 developers의 'OG 캐시 초기화' 도구를 쓰거나 잠시 후 다시 시도하세요.
